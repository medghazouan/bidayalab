TOTAL FILES AND DIRECTORIES:
- Main source files: ~50+ files
- Build files (.next): ~4900+ files (generated)

PROJECT TYPE: Next.js 14+ with App Router
FRAMEWORK: React with TypeScript
STYLING: Tailwind CSS
DATABASE: MongoDB
DEPLOYMENT: Vercel-ready

This is a professional portfolio/agency website with:
- Dynamic routing for projects and categories
- API endpoints for contact, orders, and content management
- Responsive design with Tailwind CSS
- TypeScript for type safety
- MongoDB integration for data persistence
